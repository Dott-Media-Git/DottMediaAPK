Social Integration Checklist — Dott Media

This file can be printed to PDF via your browser's Print → Save as PDF.

Common requirements

  • Active Dott Media user account with permission to add integrations.
  • Admin or appropriate role on the social account.
  • Developer app / project created for the platform when required.
  • OAuth redirect URI / callback URL configured to point to the Dott Media callback.
  • Required scopes/permissions granted during OAuth or token creation.
  • Tokens/credentials stored (access/refresh tokens, secrets, IDs).
  • Webhook callback & verify token configured when needed.
  • Two-factor auth recommended/required for some platforms.

Instagram

Prerequisites: Business/Creator account; linked to a Facebook Page; admin in Business Manager.

Permissions: instagram_basic, pages_show_list, instagram_content_publish, pages_read_engagement, instagram_manage_comments, instagram_manage_messages, instagram_manage_insights, pages_manage_metadata.

Facebook

Prerequisites: Facebook Page; admin in Business Manager; 2FA recommended.

Permissions: pages_show_list, pages_read_engagement, pages_manage_posts, pages_manage_engagement, pages_manage_metadata, pages_messaging.

WhatsApp

Meta Business Manager with verified business; WABA and verified phone number; webhook configured; permissions: whatsapp_business_messaging, whatsapp_business_management.

LinkedIn

Company Page; Super Admin or Content Admin; LinkedIn app with OAuth and Marketing API access.

TikTok

Creator/Business account; Developer app; scopes: user.info.basic, video.upload, video.publish.

X (Twitter)

Developer Project + App; OAuth 1.0a access token + secret.

Threads

Uses Instagram Business account; same prerequisites and permissions as Instagram.

YouTube

Google Cloud Project with YouTube Data API; OAuth client; scope https://www.googleapis.com/auth/youtube.upload.

Verification checklist

  • Integration shows Connected in Integrations screen.
  • Test post/upload or message succeeds.
  • Webhooks/inbound messages received when applicable.
  • Analytics/engagement metrics appear where expected.
