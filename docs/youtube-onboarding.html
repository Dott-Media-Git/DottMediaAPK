Overview

This onboarding guide explains what is required to connect a client YouTube channel to the Dott Media platform for video uploads and reporting.

Estimated time: 30-50 minutes

Account prerequisites

  • YouTube channel with upload permissions.
  • Google account owner or manager access.
  • Consent screen configured (External or Internal).

Technical prerequisites

  • Google Cloud project with YouTube Data API v3 enabled.
  • OAuth client ID and client secret created.
  • Redirect URI set to the Dott Media callback URL.

Required permission

Scope Why it is needed
https://www.googleapis.com/auth/youtube.upload Upload and publish videos to the channel.

Connection steps

  1. Create OAuth credentials in Google Cloud Console.
  2. Add the Dott Media redirect URI to the OAuth client.
  3. Open Dott Media > Integrations > YouTube, then select Connect.
  4. Approve the YouTube upload permission.
  5. Optional: set the default privacy status in Dott Media.

Data we store per user

  • YouTube refresh token (primary credential).
  • Optional access token and channel metadata.
  • Default privacy status.

Verification checklist

  • YouTube status shows Connected in the Integrations screen.
  • Test upload succeeds.
  • Channel metadata appears in reporting.
