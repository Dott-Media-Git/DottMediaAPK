# TikTok/Reels Cover Template (bwinbet)

## Canvas
- Size: 1080x1920 (9:16)
- Background: Brand Gold with green gradient or black panel

## Layout
- Headline stack: center-left, 2-5 words, large.
- Subhead: small stat line below headline.
- Accent: thin green underline or badge.
- Logo: bottom-left, width 200-240px.

## Typography
- Headline: FreightSansProSemibold, 96-140pt.
- Subhead: Roboto, 32-40pt.

## Notes
- Avoid UI zones: keep key text between 15% and 80% vertical height.
- Use black text on gold or gold text on black panel.
