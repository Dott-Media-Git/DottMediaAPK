Overview

This onboarding guide explains what is required to connect a client TikTok account to the Dott Media platform for video publishing and performance tracking.

Estimated time: 25-45 minutes

Account prerequisites

  • TikTok Creator or Business account.
  • Account owner approval for API publishing access.
  • TikTok Developer account with a live app.

Technical prerequisites

  • TikTok app with Content Posting API enabled.
  • Client key and client secret configured.
  • Redirect URI set to the Dott Media callback URL.

Required permissions

Scope Why it is needed
user.info.basic Access basic account profile information.
video.upload Upload videos for publishing.
video.publish Publish videos to the TikTok account.

Connection steps

  1. Configure the TikTok app in the Developer portal.
  2. Add the Dott Media redirect URI in the TikTok app settings.
  3. Open Dott Media > Integrations > TikTok, then select Connect.
  4. Approve the requested permissions.
  5. Return to Dott Media to confirm the connection.

Data we store per user

  • TikTok access token and refresh token.
  • Open ID and granted scopes.
  • Token expiry timestamps.

Verification checklist

  • TikTok status shows Connected in the Integrations screen.
  • Test video upload succeeds.
  • Performance metrics appear in reporting.
